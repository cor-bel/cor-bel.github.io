Der Test

Glaubenssätze können mit dem Cor-Bel-Test erfasst werden. Der Cor-Bel-Test wurde nach modernsten wissenschaftlichen Standards entwickelt. Er baut auf dem Cor-Bel-Modell auf, einem hierarchischen Modell zur Struktur von Glaubenssätzen. Der Test besteht aus 291 Items und erlaubt eine präzise und differenzierte Erfassung selbstbezogener Glaubenssätze. Die Bearbeitung des Tests dauert etwa vierzig Minuten. Der Test kann an einem Computer oder mobilen Endgerät durchgeführt werden. Es ist auch möglich, die Durchführung zu unterbrechen und zu einem späteren Zeitpunkt fortzusetzen.

Jetzt teilnehmen!

Die Durchführung des Cor-Bel-Tests kostet 490 Euro einschließlich eines Gutachtens im Umfang von 28 Seiten. Um eine Testdurchführung zu bestellen schreiben Sie bitte eine E-Mail an info@cor-bel.de. Sie erhalten zunächst eine E-Mail mit Angaben zur Bezahlung. Nach Zahlungseingang erhalten Sie einen Link zum Testverfahren. Haben Sie das Testverfahren vollständig bearbeitet erhalten Sie unmittelbar Ihr persönliches Gutachten.

Die Ergebnisse werden im Anschluss an die Durchführung in einem umfangreichen Gutachten zurückgemeldet. Im Gutachten wird zunächst beschrieben, was Glaubensätze sind und welche Glaubenssätze gemessen wurden. Anschließend werden Ihre Ergebnisse dargestellt und ausführlich erläutert. Sie erhalten dabei Hinweise zur Entstehung von Glaubenssätzen, zu Folgen für das Verhalten sowie zu Ansatzpunkten für die Veränderung. Der Test kann sowohl in Eigeninitiative durchgeführt werden als auch im Rahmen einer Beratung, Supervision oder eines Coachings.
